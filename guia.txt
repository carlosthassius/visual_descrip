#Passo 1: Inicialização

    Define parâmetros como limiar (threshold) e número de bins (80)

    Carrega tabela de quantização pré-definida (QuantTable)

#Passo 2: Pré-processamento da imagem

    Converte a imagem para tons de cinza usando a fórmula Y = 0.299R + 0.587G + 0.114B

    Normaliza os valores de intensidade para o intervalo [16.5, 235.5] (escala TV)

#Passo 3: Definição dos blocos

    Calcula o tamanho dos blocos (blockSize) baseado nas dimensões da imagem

    Divide a imagem em blocos não sobrepostos do tamanho calculado

#Passo 4: Análise de bordas em cada bloco

    Para cada bloco:

        Divide em 4 sub-blocos menores

        Calcula a intensidade média de cada sub-bloco

        Aplica 5 filtros de borda diferentes (vertical, horizontal, 2 diagonais e não-direcional)

        Determina o tipo de borda predominante com base na maior resposta aos filtros

#Passo 5: Construção do histograma

    Classifica cada bloco em uma das 5 categorias de borda ou "sem borda"

    Distribui os resultados em 16 regiões locais (4x4) da imagem

    Para cada região local, constrói um histograma com 5 bins (um para cada tipo de borda)

#Passo 6: Normalização

    Normaliza cada histograma local pelo número de blocos na respectiva região

    Concatena todos os 16 histogramas locais para formar um vetor de 80 elementos (16x5)

#Passo 7: Quantização (opcional)

    Aplica quantização aos valores do histograma usando a tabela QuantTable

    Retorna o vetor de características final
